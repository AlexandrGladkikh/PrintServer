Данный PrintServer предназначен для печати принимаемого текста на изображении с последующим выводом на принтер.
Собирается приложение вызовом make из директории bin.
Проект создан при помощи Qt Creator .pro файл расположен в корне проекта

Принцип работы: Печатает первые принимаемые данные, после чего закрывает соединение.
Аутентификация не предусмотрена так как предполагается использование в безопасной среде (планирую добавить чуть позже).
